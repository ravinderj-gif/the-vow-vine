# THE VOW VINE

Luxury cinematic wedding photography & wedding films website.

## Tech Stack

- React.js (Vite)
- Tailwind CSS
- Framer Motion + GSAP
- React Router DOM
- Swiper.js
- Lenis Smooth Scroll

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```
